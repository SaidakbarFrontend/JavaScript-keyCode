JavaScript Key Code

  • event.key
    The value of the key pressed. Acaunts for modifiers keys that return CAPS and alternate chears.
  • event.location
    Some keys exist more than once on your keyboard. This provides the location of the key pressed. Try it with both shifts.
  • event.code
    The physical key on the keyboard. Doesn't care if you are holding a modifier like Shift.
  • event.which
    even.which and even.keyCode are deprecated in modern browsers.Use .key or .code instead.
